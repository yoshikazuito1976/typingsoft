# TypingSoft

ブラウザ上で動作する日本語タイピング練習ソフトです。  
現在は **入力制御ロジックの改修を進行中** です。

---

## 開発方針（重要）

- `main` ブランチは **常に動作する状態** を保つ
- 直接 `main` には push しない
- 各自の作業は **人別ブランチ** で行う
  - 例：
    - `dev/yito`
    - `dev/partner`
- `main` への反映は **Pull Request 経由** とする

---

## 現在の作業内容

- 入力処理の整理
  - ミス時に入力を進めない制御
  - 正解文字のみを受け付ける方式に変更中
- ローマ字入力の揺らぎ対応は **後続フェーズ** とする

---

## ファイル構成（抜粋）

- `index.html`  
  画面構成・DOM定義
- `program.js`  
  タイピング処理・入力制御ロジック（主に改修対象）
- `kanaroma.js`  
  カナ → ローマ字対応表
- `style.css`  
  表示・配色・レイアウト
