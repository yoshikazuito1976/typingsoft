
# TypingSoft（学内タイピング練習用）

本リポジトリは、学内で使用するタイピング練習用Webアプリの
開発・改善・情報共有を目的としています。

## 目的
- タイピングソフトの品質改善
- 学内職員間での仕様・課題・改善案の共有
- GitHubを用いた開発フローの学内練習

## 対象外（このリポジトリに置かないもの）
- 学生・職員の個人情報
- 学内限定資料や内部事情の詳細
- サーバ設定情報、認証情報、鍵、IPアドレス
  

ブラウザ上で動作する日本語タイピング練習ソフトです。  
現在は **入力制御ロジックの改修を進行中** です。

---

## 開発方針（重要）

- `main` ブランチは **常に動作する状態** を保つ
- 直接 `main` には push しない
- 各自の作業は **人別ブランチ** で行う
  - 例：
    - `dev/yito`
    - `dev/partner`
- `main` への反映は **Pull Request 経由** とする

---

## 現在の作業内容

- 入力処理の整理
  - ミス時に入力を進めない制御
  - 正解文字のみを受け付ける方式に変更中
- ローマ字入力の揺らぎ対応は **後続フェーズ** とする

---
## Issueについて
不具合報告・改善案・メモなど、気づいたことは短文でIssueに残してください。
完璧な記述や技術的な詳細は不要です。

## ファイル構成（抜粋）

- `index.html`  
  画面構成・DOM定義
- `program.js`  
  タイピング処理・入力制御ロジック（主に改修対象）
- `kanaroma.js`  
  カナ → ローマ字対応表
- `style.css`  
  表示・配色・レイアウト
